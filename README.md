# rag-CoE

UM RAG (Retrieval-Augmented Generation) para questões sobre Código de Ética

## Setup

### Requisitos
- Python 3.14.6
- pip
- numpy== 2.4.6
- pypdf==6.13.3
- google-genai==2.8.0

### Installation

1. Clone or create the virtual environment:
```bash
python -m venv .venv
```

2. Activate the virtual environment:
   - On Windows: `venv\Scripts\activate`
   - On macOS/Linux: `source venv/bin/activate`

3. Install dependencies:
```bash
pip install -r requirements.txt
```

### Running the Project

```bash
python main.py
```
